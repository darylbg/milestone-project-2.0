# Holiday UK

This website displays holiday options in the UK

Using HTML, CSS, javascript, and jquery, this website gives users an easy interface
to select a holiday in the UK. A clean and attractive yet simple and convenient layout lets users
easily set holiday parameters and visually locate dream vacations with a hotel picker.

## UX

The UX for this project is built around minimalism and easy user interface.
The user progress in this page naturally flows down from the top starting off with a bright and bold
landing page with clear navigation. Continuing down the page has the user setting holiday specifics, picking a location, selecting a feature deal and finally seeing the componay contact info.

wireframes are located:  [here](wireframes/Book1.xlsx)

## Features

→ The UX for this project is built around minimalism and easy user interface.
  The user progress in this page naturally flows down from the top starting off with a bright and bold
  landing page with clear navigation.

→ The second section of the website lets users easily set the criteria
  for their holiday with selectors appropriate for each parameter, i.e. counting toggle to set person and room limits, checkboxes for house preferences, a calendar for date picking and a pricing slider to easily visualize expense.

→ The next section has an embedded map with a city picker for a user to easily visualize
  a holiday location. When a city is selected, a list of hotels are pinned to the map and displayed in a  side bar for simple varied selection.

→ Further down the website various featured holidays are shown with easy access for booking them.

→ Lastly the footer holds the contact info with an email sign up form to access promotion deals with the company social links listed below.

### Existing Features
- navigation - allows users to easily navigate throughout the site with links
- people and room count - lets users specify the people and room numbers with clickable counters
- housing type selector - lets users specify the house type or types to stay in with checkboxes
- date picker - gives an easy visual for picking holiday dates code customized from: http://www.daterangepicker.com/ uses jQuery, Moment.js and Date Range Picker's files custom css
- price range selector - lets a user specify the holiday price range with a slider
- embedded map - through google maps api lets users locate hotels and holiday destinations with an easy city search feature and hotel display sidebar
- preselected holiday packages - gives a user the choice for a premade holiday with easy booking
- business letter sign up - gives a user the option via email api to sign up for promotions and deals
- contact info - gives clear company contact and social media links and info
